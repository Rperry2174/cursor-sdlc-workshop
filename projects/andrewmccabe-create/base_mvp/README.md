# Base MVP — Flickshot

Angry Birds–style slingshot: drag the red bird from the sling, release to fling, knock the green pig off the stack.

## Run locally

From the repo root:

```bash
cd projects/andrewmccabe-create/base_mvp && python3 -m http.server 8080
```

Open [http://localhost:8080/](http://localhost:8080/) (or open `index.html` directly in the browser).

## Files

- `index.html` — page shell
- `styles.css` — layout and HUD
- `game.js` — canvas, physics, input
