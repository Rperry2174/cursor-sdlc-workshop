# Misty's Ultimate Coffee Shop Ranking ☕

A simple, local-first app to rate and rank your favorite coffee shops.

## How to Run

Just open `index.html` in your browser:

```bash
open index.html
```

Or ask Cursor: *"Run this project for me"*

## What It Does

- Add coffee shops with a 1–5 star rating
- See them ranked best-to-worst
- Delete shops you no longer want to track
- Data persists in your browser (localStorage)

## Planned Features

1. **Drink Tags** — tag shops with what they're known for (Latte, Espresso, Cold Brew…)
2. **Search & Filter** — find shops by name or filter by drink type
3. **Visit Counter & Favorites** — track visits and pin your favorites to the top
