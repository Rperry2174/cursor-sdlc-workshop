# Base MVP — Leaf Arena Brawl

Local two-player fighting game (see [`../prd.md`](../prd.md)). Open `index.html` in a browser.

- **P1:** `A` / `D` move · `Q` punch · `W` kick · `E` hold defend  
- **P2:** arrows move · `U` punch · `I` kick · `O` hold defend  
- **Select:** `1`–`4` for each player · **Rematch:** `R` after a KO

No build step or server required.
